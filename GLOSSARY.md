# Glossary

Canonical language for the decisions and artifacts produced by these agent skills.

**Prototype**:
A disposable, question-shaped representation that makes one consequential product-experience decision directly judgeable before production commitment.
_Avoid_: Wireframe as the umbrella term

**Production commitment**:
The point at which a confirmed direction is rewritten to delivery quality and retained in the product's main implementation path.
_Avoid_: Implementation, because disposable implementation may be part of the prototype

**Variant**:
One of a small set of alternatives that differs on the governing decision while holding confirmed constraints fixed.
_Avoid_: Mockup, option
